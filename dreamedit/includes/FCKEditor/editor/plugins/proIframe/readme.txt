proIframe 1.0 for FCKeditor
----------------------------------

proIframe 1.0  
With this plugin your able to insert iframes into the FCKeditor.
So far the following attributes is available:
width, height, scrolling, frameborder, align, id, name, style.

Written 2005 by Ronni Steen Rasmussen aka Iumentum, ronni.rasmussen@gmail.com


Installation Instructions
-------------------------

1) Unzip the proIframe folder to your FCKeditor /plugins folder
2) Add the following line to fckconfig.js (somewhere after FCKConfig.PluginsPath):

FCKConfig.Plugins.Add( 'proIframe', 'en,da' ) ;

3) Add 'proIframe' at the desired position to your FCKConfig.ToolbarSets to have the HtmlTiles button displayed.
